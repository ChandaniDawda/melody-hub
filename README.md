# 🎵 Melody-Hub
Melody Hub is a React-based music streaming app that allows users to search songs, explore artists, and play audio previews using the iTunes API. Built with modern UI and interactive features.


## 🚀 Features

- 🔍 Search songs by artist or title
- 🎧 Play and pause audio previews
- 🎵 Fetch real-time data from iTunes API
- ⚡ Fast and responsive UI using React
- 🎯 Only one song plays at a time

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- iTunes API
